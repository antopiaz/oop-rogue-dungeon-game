## Project Requirements

* Must use and identify 5 design patterns
* You should use dependency injection where possible
* Code to abstractions
* Treat things polymorphically: no big if-then-else or switch statements
* Must have either a UI or some form of persisted state (ORM, file, database)

### Design patterns
* Factory for items, creatures
* Builder for dungeon and rooms
* Decorator for equipping items
* Strategy for creature's game tree
* Facade for game setup and playing
